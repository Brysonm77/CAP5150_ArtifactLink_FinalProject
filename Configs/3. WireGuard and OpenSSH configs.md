## WireGuard (WG)
### Key-Gen
#### Ubuntu server
* Key generation 
  * cd /etc/wireguard
  * sudo umask 077
    * permissions given to the owner
  * sudo wg genkey | tee server_private.key | wg pubkey > server_public.key
    * creates a private key for WG
   
* Deploying WG configs

    * sudo systemctl restart wg-quick@wg0
